# Murattel

A Quran player with real-time verse display, featuring recitations by renowned reciters in the Hafs an Asim narration.

---

## Tech Stack

- **SolidStart** — main framework
- **SolidJS** — state management and reactivity
- **TypeScript**
- **CSS** — responsive design with no external UI libraries
- **mp3quran.net API** — audio files and verse timing data

---

## Current Features

- Full playback of all 114 surahs
- Reciter selection from a curated list of Hafs an Asim reciters
- Real-time verse display synchronized with audio playback
- Interactive progress bar with time display
- Play, pause, next, previous controls
- Repeat and shuffle modes
- Fully responsive across all screen sizes
- Custom 404 page

---

## Getting Started

```bash
git clone https://github.com/your-username/murattel
cd murattel
npm install
npm run dev
```
